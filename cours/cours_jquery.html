JQuery

Il s'agit d'une librairie externe. Il faut donc l'ajouter si l'on veut pouvoir l'utiliser. Les méthodes principales pour ajouter JQuery sont :
  • télécharger la librairie (css + JS) et les importer manuellement
  • utiliser un CDN
  • utiliser un géstionnaire de dépendances (npm, yarn, ... ; sera vu en Angular)

Différence CDN/local

En local, on ajoute les sources dans notre code (téléchargement manuel). CDN (Content Delivery Network), en gros le code est disponible sur un serveur d'où on le télécharge (balise script, link). La suptilité du CDN est qu'il est répliqué avec de noombreuses instances (un peu partout dans le monde).

Installation

Local

  • on cherche/trouve la librairie avec la version souhaité
  • on télécharge les fichiers (CSS, JS, ...)
  • on place ces fichiers dans le projet (arborescence fichier)
  • on mets les liens manuellement (link, script)

CDN

  • on cherche/trouve le lien CDN pour la librairie et version souhaité
  • on mets balise (link/script) avec ce lien

Début

        
            // On peut appeler toutes les fonctions de jQuery a partir de l'objet:
            jQuery

            // On a un alias (raccourci):
            $

            // Recuperation d'un element du DOM
            $('')
            $('body')

            // Creation d'une nouvelle balise
            $('')
            $('body').prepend($('').addClass('dedez').attr('href', 'http://www.google.fr'))

            // Encapsulation d'un objet JS en jQuery (permet de modifier les methodes a utiliser)
            $(document)

            // Gestion d'evenement.
            $(document).on('click', _ => alert('Salut, vois comment je suis pas du tout relou'))
