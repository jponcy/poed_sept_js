ES6 (2015)

Pas encore suporté à 100% par les navigateurs. On utilise souvent la transpilation pour récupérer un code ES5 plus fonctionnel (portable). Principale différences :

  • ajout du mot clef class (Java like)
  • ajout des visibilités (public, protected, private)
  • ajout de constructeur de classe (constructor)
  • héritage (Java like)
  • ajout let (à la place de var ; plus de soucis de post déclarations)
  • ajout de const
  • meilleur support des arrow functions
  • ajout getter/setter (get/set ; exemple : get value() { return this.instanceValue; } // pourra être appelé avec this.value;
  • ajout système paramètre par défaut
  • ajout paramètre infini (myfn(... values) { values.forEach(console.log) } // pourra être appelé avec this.myfn(1, 2, 3)
  • destructuration
  • générateur (yield)
            
class Human {
    speed = 3;

    move() {
        console.log(`Je me déplace à ${this.speed} km/h`);
    }
}

class Gladiator extends Human {
    speed = 54321;
}
